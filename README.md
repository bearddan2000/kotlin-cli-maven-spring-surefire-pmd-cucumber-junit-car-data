# kotlin-cli-maven-spring-surefire-pmd-cucumber-junit-car-data

## Description
A POC for spring app using junit5, cucumber,
and pmd framework with surefire plugins.

PMD analyze source code for potential bugs.

## Tech stack
- kotlin
- maven
  - spring
  - junit5
  - cucumber
  - surefire
  - pmd

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
